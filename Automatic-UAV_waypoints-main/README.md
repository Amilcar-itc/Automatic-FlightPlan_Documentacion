# IA-REH
## Desarrollo de un modelo de IA para el reconociemiento de estrés hídrico en cultivos de tomate
### Estudiante: Josué Celaya Gerardo
### Maestro/supervisor: Zuriel Dathan Mora Félix

## Propósito del repositorio
Aquí se subirán los avances del desarrllo de el algoritmo para el reconocimiento de plantas con estrés hídrico en sembradíos de tomate cherry usando el Índice de Vegetación de Diferencia Normalizada (NDVI), así como la creación de puntos de coordenadas para cada una en base a imágenes multiespectrales para la posterior creación de waypoints para la creación de un plan de vuelo para drones.
